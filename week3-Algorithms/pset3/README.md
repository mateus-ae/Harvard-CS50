## Description of the problems:

### Plurality
Implement a program that runs a plurality election, per the below.

>$ ./plurality Alice Bob Charlie
>
>Number of voters: 4
>
>Vote: Alice
>
>Vote: Bob
>
>Vote: Charlie
>
>Vote: Alice
>
>Alice

[Full Problem Description](https://cs50.harvard.edu/x/2021/psets/3/plurality/)
